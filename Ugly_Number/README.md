# Ugly Numbers
Ugly Number的定義為：該數之質因數只能有 2或 3 或 5不能有其他的質因數
當然了，依照慣例，1 也算是 Ugly Number。
在此列舉一串數列：
1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15
這些就是前 11 個 Ugly Numbers。
請寫一個程式求出第N個Ugly Number。
 
## Input
第一行為測資筆數K
第二行開始為正整數N,N<=1000

## Output
每行一個 輸出K行
第N個Ugly Number

## Sample input
3
7
11
150

## Sample output
8
15
5832
