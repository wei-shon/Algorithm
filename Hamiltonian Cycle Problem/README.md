# Hamiltonian Cycle Problem
漢米爾頓迴路的定義為在一個無向圖上，是否存在著一個路徑可以經過圖上的所有點恰好一次，並且最後可以回到原點。
## Input
第一行為測資組數
每組第一行為圖中所含的邊的個數
每組第二行開始為邊
## Output
如果給定的圖存在漢米爾頓迴路則輸出True
反之輸出False
## Sample input
    2
    6
    (1,4)
    (1,5)
    (2,3)
    (2,5)
    (3,4)
    (3,5)
    6
    (1,2)
    (2,3)
    (3,4)
    (3,6)
    (4,5)
    (5,6)

## Sample output
    True
    False
