# Relational Operators
有些運算子（operator）是用來檢查兩個數值之間的關係，這種運算子稱為關係運算子（relational operators）。 給你兩個數值你的工作就是要找出它們之間的關係是(1)第一個大於第二個 (2)第二個小於第一個 (3)兩個一樣大。

## Input
輸入的每列有兩個整數 a 和 b （ |a|,|b| < 1000000001）。
如果第一個整數 a = 0，則終止程式。

## Output
對於每組測試資料，輸出，">", "<" 或 "="，代表該二數字的關係。

## Sample Input
10 20
20 10
10 10
0

## Sample Output
<
>
=
