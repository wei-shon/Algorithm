# Insertion sort
## Input
每一筆的測資都會有兩行，第一行為總長度L (0 ≤ L ≤ 50)，
第二行為從1到L的排列組合
現在要將此序列由小到大重新排列。
若第一行輸入L = 0時，則結束程式
## Output
每組測資請輸出”Optimal swapping takes S swaps.” S為swap次數
## Sample Input 
3
1 3 2
4
4 3 2 1
2
2 1
0
## Sample Output 
Optimal swapping takes 1 swaps.
Optimal swapping takes 6 swaps.
Optimal swapping takes 1 swaps.
